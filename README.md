# Gender Predictor

## Getting started

1. Create virtualenv

```bash
   # create env
   conda create --name gender

   # activate it
   conda activate gender
```

2. install dependencies

```bash
  pip install -r requirements.text
```

3. run app

```bash
 uvicorn app:app --reload
```

## Built with

1. FastAPI
2. Sklearn
3. Tailwindcss
